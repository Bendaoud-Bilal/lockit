# Équipe Lockit Password Manager

## Workflow
- Chaque développeur travaille sur sa branche de fonctionnalité
- Une fois terminé, merge/push vers `dev` pour intégration
- Une fois que tout fonctionne correctement sur `dev`, on push vers `main` (release)

## Assignations

| Fonctionnalité | Branche | Responsable(s) |
|----------------|---------|----------------|
| Authentication & Sécurité | auth-feature | rymene |
| Vault (Gestionnaire MP) | vault-feature | abdelawal + melissa |
| 2FA Authenticator | authenticator-feature | ania |
| Dashboard | dashboard-feature | bilal |
| Send & Folders | send-folders-feature | mohammed |

## Branches principales

| Branche | Objectif | Règles |
|---------|----------|--------|
| dev | Intégration et test | Merge depuis les branches de features |
| main | Production/Release | Merge depuis dev uniquement |
